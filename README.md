# Выполнение домашней работы № 18: REST API Часть 2

Команда для установки зависимостей: pip install -r requirements.txt  
Перед запуском тестов создаем .env-файл в корне проекта такого формата:  
MY_PASSWORD=<ваш пароль>  
MY_EMAIL=<ваш email>  
NAME_SENDER=<имя отправителя>  
NAME_RECIPIENT=<имя получателя>  
Команда для запуска тестов: pytest tests_api  
Команда для формирования отчета: allure serve allure-results
